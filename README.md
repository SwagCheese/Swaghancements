
# Swaghancements #  

## What is Swaghancements? ##
Swaghancements is a free and open source Minecraft 1.8.9 client based off of Hyperium Client. It combines many popular features and "enhancements" in order to provide a unique experience designed to give you an advantage in pvp.

The client is currently in development.

## Features ##
- Discord Rich Presence (Custom)
- Old Animations (Custom)
- Reach Display (Custom)
- Togglesprint (Custom)

### Addons ###
Addons are Hyperium's system of mods, like Forge, except they're done using the [Addon Workspace / Addon API](https://github.com/HyperiumClient/Addon-Workspace/). The selection to currently available addons is low, so if you wish to make some, feel free to share it with other Hyperium members.
